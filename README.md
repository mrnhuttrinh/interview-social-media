Create React App is divided into two step:

#1 Install packages
npm:
```
npm install
```
yarn: Should be use this on window.
```
yarn install
```
#2 Run app
```
npm start
```

```
app
├── README.md
├── node_modules
├── package.json
├── .gitignore
├── public
```
